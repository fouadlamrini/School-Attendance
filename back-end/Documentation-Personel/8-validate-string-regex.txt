1️⃣ chno howa had star?
had star kytst3ml bach nt7a9a9 mn  sigha dyal text (string) kankhdmo b regular expression (RegEx).

^\d{4}-\d{2}-\d{2}$ = pattern dyal date bsigha dyal YYYY-MM-DD

2️⃣ char7 dyal pattern ^\d{4}-\d{2}-\d{2}$:

^ = lbdya dyal string
\d{4} = 4 arqam (0-9) katmthal l3am (YYYY)
- = dash katfr9 bin l3am w chhr
\d{2} = 2 arqam katmthal lchhr (MM)
- = dash katfr9 bin lchhr w nhar
\d{2} = 2 arqam katmthal lnhar (DD)
$ = nihaya dyal string
✅  exemple sa7i7: "2025-12-26"
❌  exemple khata2: "25-12-26" wla "2025/12/26"
==========================================================
