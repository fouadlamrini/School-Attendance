2️⃣ char7 dyal pattern /^[^@\s]+@[^@\s]+\.[^@\s]+$/:
^ = lbdya dyal string
[^@\s]+ = ay 7aja machi @ w machi spaces, katmthal smiya dyal luser (local part)
@ = katfr9 bin smiya dyal luser w smiya dyal domain
[^@\s]+ = ay 7aja machi @ w machi spaces, katmthal smiya dyal domain
\. = dot katfr9 bin smiya dyal domain w tld
[^@\s]+ = ay 7aja machi @ w machi spaces, katmthal tld (top-level domain) b7al com, net, org...
$ = nihaya dyal string
✅ exemple sa7i7: "student@example.com"
❌ exemple khata2: "student@ example.com" wla "studentexample.com"
