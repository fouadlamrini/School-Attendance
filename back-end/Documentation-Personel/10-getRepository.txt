notion getRepository
============================================
1️⃣ chno howa getRepository?
getRepository hiya method man TypeORM (DataSource) kaymaknak bach tat3aml m3a database bsohola 3la mostawa entity mo3ayan.

Repository = object fih jmi3 operation asasiya (CRUD)  lkhasa b entity.

kola entity 3ndo repository mosta9il.

f TypeORM 0.3.x، knst3ml DataSource instance  bach njib repository، machi mobachara mn typeorm.
=============================================
2️⃣ Syntax général
import { AppDataSource } from '../data-source';
import { entityClass } from '../entities/entityClass';

const entityClasseRepo = AppDataSource.getRepository(entityClass);
entityClass:smiya dyal class dyal entities dyawli (model)
=============================================
